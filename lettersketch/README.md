# LetterSketch
Test of writing tools

## Requirements
```
  pip install flask
  pip install flask-cors
```

## Running

Python 2:

```
  python lettersketch_looging.py
  python -m SimpleHTTPServer
```
Python 3:
```
  python lettersketch_looging.py
  python -m http.server 8000
```

Open your web browser to [http://localhost:8000/](http://localhost:8000/).
